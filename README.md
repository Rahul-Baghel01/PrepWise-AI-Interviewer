PrepWise is an AI-powered mock interview platform that helps students and job seekers prepare for technical and HR interviews through realistic voice-based conversations. The platform generates personalized interview questions based on the selected role and technology stack, conducts interactive interviews using AI voice agents, and provides instant feedback to improve communication, technical knowledge, and interview confidence.

The application integrates Firebase for authentication and data storage, Gemini/OpenAI for intelligent question generation and evaluation, and Vapi AI for real-time voice interactions. It also stores interview history, performance metrics, and feedback, allowing users to track their progress over time.

Key Features
AI-generated interview questions tailored to specific roles and skills.
Real-time voice interviews using AI voice agents.
Personalized feedback with performance analysis.
User authentication and secure interview history using Firebase.
Dashboard to manage previous interviews and monitor progress.
Responsive modern UI built with Next.js and Tailwind CSS.
